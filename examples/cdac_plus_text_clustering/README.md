## CDAC+ 文本聚类算法

```text
基于大量未标注的句子, 和部分已标注的句子. 
模型需要在考虑到已标注的句子的聚类情况, 对未标注的句子做聚类. 

此方法可用于
1. 新意图发现. 
2. 同义句 

```

```text
参考链接
https://github.com/thuiar/CDAC-plus
https://arxiv.org/abs/1911.08891

```

```text
算法思路: 

1. classification. 
因为此处有一百多个类别, 而论文中的数据集只有十几, 二十几个类别. 
因此, 我设置了一个单纯分类训练的过程, 使各类别先进行一定程度的聚集. 
否则的话, 在 pretrain 时, 由于类别样本的相似程度不够, 它可能无法很好地训练. 
冻结 transformer 除最后一层的所有参数. 进行训练. 

2. pretrain. 
(1)标注数据上的聚类, 
(2)全部数据上的半监督聚类. 
也可以将 classification 取消, 在 pretrain 中, 将分类与聚类训练交叉执行. 
冻结 transformer 除最后一层的所有参数. 进行训练. 

3. finetune. 
(1)kmeans 聚类, 给每个样本分类伪标签. 
(2)根据聚类分配的标签, 计算 `辅助目标分布`, `学生分布`, 优化 KL 散度. 实现聚类. 


```
